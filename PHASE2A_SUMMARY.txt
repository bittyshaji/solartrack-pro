SERVICE FUNCTIONS

SEARCH SERVICE (searchService.js):
1. performFullTextSearch() - Multi-type full-text search
2. performProjectSearch() - Search projects with filters
3. performCustomerSearch() - Search customers with filters
4. performInvoiceSearch() - Search invoices with filters
5. getSearchHistory() - Retrieve user's search history
6. saveSearch() - Log searches for history/analytics
7. clearSearchHistory() - Delete all search history
8. getAutocompleteSuggestions() - Autocomplete suggestions
9. getSearchFilters() - Available filters per type
10. advancedSearch() - Complex search with pagination

FILTER SERVICE (filterService.js):
1. createSavedFilter() - Create and save custom filter
2. getSavedFilters() - Retrieve user's filters
3. updateSavedFilter() - Modify filter configuration
4. deleteSavedFilter() - Remove filter
5. setDefaultFilter() - Mark filter as default
6. applyFilter() - Client-side filter application
7. validateFilterConfig() - Validate filter structure
8. buildFilterQuery() - Convert filter to SQL WHERE clause
9. getFilterPresets() - Common filter templates
10. combineFilters() - Combine multiple filters (AND logic)
11. getDefaultFilter() - Get user's default filter
12. cloneFilter() - Duplicate filter
13. exportFilter() - Export filter as JSON
14. importFilter() - Import filter from JSON

  Columns: id, user_id, search_term, search_type, result_count, created_at

ENHANCED TABLES:

- customers: Added search_vector (TSVECTOR) with trigger

INDEXES:
- GIN index on projects.search_vector for fast FTS

- GIN index on invoices.search_vector for fast FTS
- Indexes on search_logs for user_id, created_at, search_term
- Indexes on saved_filters for user_id, type, is_default
